# tzbb
>一个快速搭建ReactNative移动应用脚手架
## 依赖包
tzbbLibs/workbench 脚手架基础底座包
地址：https://github.com/dreamsleep11/tzbbLibs
## 目录介绍
1. android
android基础项目目录
2. ios
ios基础项目目录
3. js
js基础代码目录
4. js/routes 
路由配置目录
5. js/src
业务代码目录，app开发主要代码写在这里
6. js/res 
静态资源文件包
7. js/src/pages 
页面代码包
8. js/src/Themes
主题代码包
9. js/src/cores
核心代码包（业务开发基本不会更改）
10. js/src/main
入口包
